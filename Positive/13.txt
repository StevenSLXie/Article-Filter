交通运输部紧急通知，要求各省交通厅制定具体实施方案，以保障国庆节假日期间小型客车免费通行。节日将近，应者寥寥。
&nbsp;
这让我想起一封著名电报。清政府引拳匪入室于前，向列强宣战于后，令各省督抚带兵勤王。李鸿章深知，积贫积弱，不堪一战，于是复电朝廷：“此乱命也，粤不奉诏。”
&nbsp;
下午，在中国国际广播电台评论高速公路免费，意犹未尽，还是写下来清楚一些，也便将来验证。
&nbsp;
一、通知是否合法？
&nbsp;
高速公路公司有股东，惟他们有权利决定是否免费。
&nbsp;
有些高速公路公司，股权由某省国资委持有，从法律上讲，需要该省人大授权，该省国资委才可以决定免费——交通运输部越权了。
&nbsp;
还有一些高速公路公司，股权由企业、个人持有，不少企业还在A股上市。从法律上讲，惟他们有权利决定是否免费——交通运输部侵权了。
&nbsp;
交通运输部如此通知，相当于卫生部通知要求医院免费看病，农业部通知要求饭店免费吃饭——需要很大勇气。
&nbsp;
有人会说：“又不是全年、对所有车免费，用不着上纲上线吧？”限制政府权力，尤其是限制中央政府权力，关乎宪政，必须防微杜渐。
&nbsp;
二、通知是否合理？
&nbsp;
高速公路公司本来可以收费，现在交通运输部要求免费。即使是遇到战争、天灾，政府征用财产，也得合理补偿。那补偿吗？
&nbsp;
如果不补偿，就是交通运输部请客，地方、企业买单。以别人钱财，谋自己政绩，此非为政之道。
&nbsp;
如果补偿，钱从何来？政府收入大部分来自税收，向不开车的人征税，补贴开车的人，这公平吗？
&nbsp;
三、通知是否能够得到贯彻？
&nbsp;
孔子曰：“君使臣以礼，臣事君以忠。”通知贯彻情况，不问便知——阳奉阴违。
&nbsp;
往常节假日期间，收费多于平时，畅通就是效益，高速公路公司自然会提前准备、全力以赴：修路安排在节前、节后，工作人员无故不得请假。谁敢懈怠，饭碗难保。
&nbsp;
一旦免费，高速公路公司可以修路、准假：路上有坑，需要限行、限速，谁也不敢解除限制，否则出事故谁负责；工作人员可以请病假——带病上岗，出事故谁负责？
&nbsp;
“不出事故就是胜利！”俗话说：“十次事故九次快，还有一次是意外。”高速公路公司会千方百计限行、限速——这还是高速公路吗？
&nbsp;
高速公路公司用免费了、无事故敷衍交通运输部，用慢、堵恶心交通运输部——这就叫无声的抗议。
&nbsp;
四、贯彻后果如何？
&nbsp;
据称，交通运输部紧急通知，初衷是鼓励出行、拉动内需。可惜，评价政策好坏，看的不是初衷，而是程序是否合法、后果是否如愿。
&nbsp;
就算通知得到贯彻，后果又会如何？免费的只是小型客车，但是这已经足以让某些路段瘫痪。广州在亚运会前对地铁免费，地铁瘫痪。北京机场高速，出城收费，一般不堵，返城免费，极少不堵。经济规律是无情的，总在呈现事与愿违。真想拉动内需，我这儿有秘诀，就四个字：减税、少管！
&nbsp;
汽车长龙堵在路上动弹不得，虽然不能鼓励出行，但是能够拉动内需——路边的农民可以翻越护栏，到路边卖方便面、矿泉水什么的。
&nbsp;
如果早产的孕妇、危重的病人被堵在路上，就得跑步前进、警车开道甚至出动直升机了。到时，还可以拍《感动高速公路》《最美交警》——这就叫折腾。
&nbsp;
时间就是金钱，高速公路就是用金钱换时间——免费就是浪费。
&nbsp;
五、民怨何来？
&nbsp;
怨言一：“美国能高速公路免费，为什么中国不能？”
&nbsp;
美国免费也有问题，但是还勉强说得过去：美国大多数家庭有车，政府向所有家庭征税，再用税款修路，免费服务占大多数的有车家庭。中国有车的家庭是少数，绝不能向穷人征税补贴富人。
&nbsp;
怨言二：“某些高速公路说好只收费十年的，为什么还要逾期收费？”
&nbsp;
我支持逾期收费。公司与政府签订BOT（修建、运营、转让）协议，出钱修路，并取得十年的收费权。十年后，政府如何处置高速公路？
&nbsp;
上策，再次拍卖给公司，收入分给民众，或者相应减税，公司则继续收费。
&nbsp;
中策，政府收回，直接运营，继续收费。国营企业弊病众所周知，不再赘述。
&nbsp;
下策，政府收回，征税维持运营，免费——这相当于不开车的人补贴开车的人，不公平。
&nbsp;
当然，还有下下策：政府收回，放弃运营，不养路，无救援，把高速公路变为坟场。
&nbsp;
怨言三：“我们车主容易吗，油价那么高，税费那么高，为什么还要交买路钱？”
&nbsp;
冤有头，债有主，前面右转是政府。中国油价很复杂，既有补贴，又有垄断利润——反行政垄断去吧！税费高，请呼吁限制权力、减免税费。出门遇流氓，回家打老婆，这不好吧……
&nbsp;
怨言四：“高速公路公司暴利，为什么不能出点儿血？”
&nbsp;
俗话说：“只见贼吃肉，没见贼挨打。”高速公路公司的运营利润率确实高，可当初修路、买路花钱也多啊！要想从根本上降低高速公路收费，就必须打破行政垄断，引入民间资本。
&nbsp;
我也不希望节假日期间交通混乱，但是这一次难免。希望交通运输部能够自重、自省，尊重宪法、法律、合同，尊重地方政府，尊重经济规律。
